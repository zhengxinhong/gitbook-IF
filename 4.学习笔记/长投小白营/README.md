

* 我看过“秘密”这个电影，这个电影讲的是吸引力法则，讲述的是如何利用吸引力法则来实现你的愿望，分为三步骤：1.向宇宙下订单（你的愿望是什么？），这个愿望越具体越好，比如我要赚100万这远比我要拥有一部小车来的更具体；2.你要坚定信念你已经拥有或已经成为你愿望里的人或事，最好能将其“视觉化”--观想；3.接受现实，然后下定决定去做。这个过程就是：成为--做--拥有
